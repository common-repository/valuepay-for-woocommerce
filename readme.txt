Accept payment on WooCommerce using ValuePay.

== Description ==

Allows customer to made payment on WooCommerce using ValuePay.

= Notes: =
- It is required to have name, email and phone number field on WooCommerce checkout page. So, if you are using any checkout simplify plugin, eg: Cartflows, please make sure that you display the required fields.
- Recurring payment only creates one payment record in WooCommerce with "On Hold" status.

== Installation ==

1. Log in to your WordPress admin.
2. Search plugins "ValuePay for WooCommerce" and click "Install Now".
3. Activate the plugin.
4. Navigate to "Plugins" in the sidebar, then find "ValuePay for WooCommerce".
5. Click "Settings" link to access the plugin settings page.
6. Follow the instructions and update the plugin settings.
